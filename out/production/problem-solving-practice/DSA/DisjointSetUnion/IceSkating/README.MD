Input:
- n in 1..100: number of snow drifts
- For each n lines:
    - x, y: coordinates of i-th snow drift
    - All location are distinct

Output:
- Minimal number of snow drifts that need to be created, to reach any snow drift from any other one.

Example:
Input:
2
2 1
1 2

Output:
1
